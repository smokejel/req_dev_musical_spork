# Changelog - Requirements Extraction Skill

## [1.0.0] - 2025-10-30

### Added
- Initial release of requirements-extraction skill
- Comprehensive methodology for extracting requirements from specifications
- Support for multiple document formats (.txt, .docx, .pdf)
- Requirement type identification (FUNC, PERF, CONS, INTF)
- Metadata extraction (section, priority, stakeholder)
- ID assignment strategy
- Quality checks for atomic requirements
- Good vs. bad examples
- Compatible with Phase 1 (Extract Node)

### Design Decisions
- Focus on atomic, testable requirements
- Metadata extraction for traceability
- Support for unstructured input documents
- JSON output format for easy parsing
